
I used the robin hood api, to display the crypto that are currently tradable on Robinhood. 

Prerequiste: 
Run requirment.txt for dependencies or as a reference for indiviual installs and version. 

Running Guide 
This app runs on localhost:8000

1. Run --> index_CryptoInfo.py to index external api to db
2. Run makemigrations
3. Run migrations
4. Run runserver

Testing 
1. Complete above
2. Run test robinhood_app
